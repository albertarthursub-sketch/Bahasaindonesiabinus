Perfect for Grades 1-2 students learning basic vocabulary with visual recognition.

### List 1: Basic Household Items

| Bahasa Indonesia | English | Notes |
|---|---|---|
| sepatu | shoe | Common, easy to find images |
| meja | table | Familiar from classroom |
| kursi | chair | Classroom furniture |
| pintu | door | Present in every room |
| jendela | window | Easy to identify |
| buku | book | School supply |

| tas | bag | Students have backpacks |

**How to Create:**
1. Go to Teacher Dashboard → Create List
2. Mode: Image Vocabulary
3. Title: "Barang Sehari-hari (Daily Objects)"
4. For each item:
   - Name: `sepatu`
   - Image: Photo of a shoe
   - Add Item
5. Save and share code with students

## Tips for Creating Image Lists

### Image Quality
- **Resolution:** 400x400px or larger
- **Format:** JPG or PNG
- **File Size:** Under 2MB for optimal loading
- **Quality:** Clear, well-lit, recognizable

### Image Selection
- ✅ Use high-contrast images
- ✅ Clear subject (object clearly visible)
- ✅ Age-appropriate
- ✅ Diverse representations (different styles/colors)
- ❌ Avoid cluttered backgrounds
- ❌ Avoid text overlays
- ❌ Avoid copyrighted material

## Pronunciation Guide

Using Web Speech API with Indonesian settings:

| Word | Pronunciation | Notes |
|---|---|---|
| sepatu | /sə-PA-tu/ | Stress on 2nd syllable |
| meja | /ME-ja/ | Clear "j" sound like in "judge" |
| kursi | /KUR-si/ | Roll the R slightly |
| pintu | /PIN-tu/ | Quick "i" sound |
| buku | /BU-ku/ | Both "u"s like in "book" |
| pensil | /PEN-sil/ | Like English "pencil" |
| tas | /tas/ | Like English "as" |
| rumah | /RU-mah/ | Soft H sound |
| warna | /WAR-na/ | W like "v" sound |
| bunga | /BUNG-a/ | Ng sound as in "singing" |

## Assessment Ideas

**Quick Checks:**
- Student can identify 8/10 objects ✓
- Student can pronounce names clearly ✓
- Student participates in group activity ✓

**Extended Practice:**
- Teacher shows image, student names it
- Student draws image and labels in Indonesian
- Matching game: image to written word
- Rhyming words in Indonesian

**Cross-Curricular:**
- Counting objects (Math)
- Colors and sorting (Math)
- Writing sentences about objects (Language)
- Drawing and describing (Art)
